# TRWNS (Train Radio Warning/Notification System)

## Install
1. Run `git clone github.com/16BitDoggo/TRWNS`. This downloads the files you need from the GitHub repo.

2. Run `cd TRWNS`. This moves you into the folder that contains the files you just downloaded.

3. `install.sh`. This automagically installs everything needed to run the script. (you may need to enter your password)

Now you have downloaded and installed everything necessary to use TRWNS.

Run `python3 trwns.py` to use the program
